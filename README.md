## 🔧 Using defender_overview

Once the app is open, it will display key information about Microsoft Defender on your PC. Here is what you can expect:

- **Real-time Protection Status:** Shows if Defender is actively protecting your computer.
- **Scan History:** Details recent scans and their results.
- **Virus & Threat Protection:** Information about current virus definitions and threat detections.
- **Firewall Status:** Indicates if Windows Defender Firewall is on.
- **Update Status:** Shows when Defender last updated itself.

The app uses clear labels and simple layouts so you can quickly understand your security level.
